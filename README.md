## Описание

Консольное приложение на C# с арифметическим калькулятором.  
Каждая операция реализована в отдельной ветке Git.

## Ветки

| Ветка | Операция |
|-------|----------|
| `feature/add`      | Сложение (+) |
| `feature/subtract` | Вычитание (−) |
| `feature/multiply` | Умножение (×) |
| `feature/divide`   | Деление (÷) |
| `final`            | Слияние всех операций |
| `master`           | Финальная версия |

## Запуск

```
dotnet run
```
или открыть MusicCalcApp.csproj в Visual Studio.
